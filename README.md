# PlantNet Flutter App

This is a Flutter application that uses the PlantNet API to identify plants from images.

## Features

- Select an image from the device's gallery.
- Choose the organ of the plant to be identified (leaf, flower, fruit, bark, or auto).
- Sends the image to the PlantNet API for identification.
- Displays a list of possible identifications with their scientific name, common names, score, and related images.
